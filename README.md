# docker-compose

# Dockerfile

Cria uma image a partir do Dockerfile
```
docker build -t python-stats-alpine .
```

# docker-compose.yml

### para subir o container
```
docker compose up -d
```

### para ver as logs
```
docker compose logs 
```

### parar o container
```
docker compose down
```
